El Cerebro del Proceso: Fundamentos del Control PID

El controlador Proporcional-Integral-Derivativo (PID) es el algoritmo más crucial y extendido en la automatización. Es el cerebro que permite a un sistema no solo medir una variable, sino mantenerla estable en un punto deseado (Setpoint). Para entenderlo, debemos analizar a los tres "expertos" que lo componen.

1. El Experto Proporcional (P)

Es la acción principal y más intuitiva. Su lógica es simple: "La fuerza de mi respuesta es proporcional al tamaño del error actual". Un error grande provoca una acción fuerte, y un error pequeño, una acción suave. Su gran defecto es el "Offset": para mantener una acción correctiva, necesita que siempre exista un pequeño error, por lo que nunca llega a la perfección.

Salida_P = Kp * Error(t)

2. El Experto Integral (I)

Este experto existe para corregir la imperfección del Proporcional. Su misión es acumular el error a lo largo del tiempo. Mientras exista ese pequeño "Offset", el Integral irá sumando fuerza de manera paulatina hasta que el error sea exactamente cero. Es el responsable de la precisión del sistema.

Salida_I = (Kp / τi) * ∫Error(t)dt

3. El Experto Derivativo (D)

No le importa el error actual ni el pasado, sino el futuro. Analiza la velocidad de cambio del error. Si la variable se acerca demasiado rápido al Setpoint, el Derivativo actúa como un freno para evitar pasarse de largo (overshoot). Aporta estabilidad y suavidad.

Salida_D = Kp * τd * d(Error(t))/dt
